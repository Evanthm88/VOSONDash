VOSON Dashboard

VOSON Dashboard is an R/Shiny application for collecting and analysing networks and associated text data. VOSON Dashboard builds on a number of R packages, in particular igraph (for network analysis) and VOSON SML. VOSON SML is an R package that provides a suite of tools for collecting and constructing networks from social media data. It provides easy-to-use functions for collecting data across popular platforms (Facebook, Twitter, and YouTube) and generating different types of networks for analysis.
